# Pocket Vault Product Goal

## Product Definition
Pocket Vault is a Base-native USDC pocket-money savings product for college and university students who are 18+ and starting to use crypto. A student creates a named vault for one specific emergency or student-life goal, deposits over time, and can only withdraw when the vault's unlock rule allows it.

## Product Promise
Save pocket money in crypto. Lock it with clear rules. Keep emergency savings harder to break casually.

## Core Thesis
Most student saving tools fail because they are too easy to break, too generic, or too close to spending money. Pocket Vault should make small crypto savings feel:

- intentional
- protected
- visible
- hard to undo
- personally meaningful

The moat is not complex finance mechanics. It is the combination of:

- narrow scope
- student-specific emotional framing
- beautiful product design
- credible onchain execution
- friction applied in the right places

## What The Product Is
- A programmable student savings vault
- A goal-first stablecoin product for pocket-money savings
- A behavior-shaping financial tool for emergency reserves
- A calm onchain consumer app for crypto beginners

## What The Product Is Not
- Not a DeFi dashboard
- Not a trading interface
- Not a lending app
- Not a multi-asset portfolio manager
- Not a noisy crypto product
- Not a public student social product

## MVP Boundaries
- Chain: Base
- Asset: USDC only
- Wallet-first onboarding
- One owner per vault
- Private vault metadata by default
- No yield
- No swaps
- No multi-chain
- No social layer
- No AI features
- 18+ student positioning only

## MVP Must Do Five Things
1. User creates a vault.
2. User deposits USDC.
3. User sees progress toward a goal.
4. User cannot withdraw until the rule allows.
5. User can withdraw only when the condition is satisfied.

## Release Order
- `v1`: time lock only
- `v1.5`: cooldown unlock
- `v2`: guardian approval

## Product Principles
- Goal-first, never protocol-first
- Calm, not noisy
- Intentional friction
- Premium simplicity
- Trust through clarity

## Core Emotional Hook
A visible, rule-protected promise to your student future self.

## Core Product Advantage
It does one thing extremely well: protect student pocket-money savings from impulse spending.
